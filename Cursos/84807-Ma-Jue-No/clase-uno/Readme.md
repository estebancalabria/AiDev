# Desarrollo con LLM

* Empecemos con Claude
    * https://claude.ai/new
    * Hoy considerado uno de los mejores modelos para programar
    * Desde su interfaz web tiene artefactos
        * Separa bien el Codigo de la respuesta del LLM
        * Se puede compartir
    * Vamos con un primer ejemplo
 
```
Quiero que me desarroles el juego pong en un unico ARTEFACTO jacascript y HTML donde el jugador de la derecha se mueva con las teclas de cursor y el de la izquierda con la A y la Z. El juego se debe poder pausar con la barra espaciadora con la  opcion de reiniciarlo o continuar. Igualmente se debe presionar la barra espaciadora para comenzar.  Queiro que el juego tenga una estetica vaporwave, con efectos de particulas y sonido, que sea moderno y visualmente atractivo.
```
